\abstract{While medical vision foundation models achieve high aggregate accuracy in ophthalmic diagnostics, their clinical deployment is severely bottlenecked by demographic biases inherited from skewed training distributions. Current mitigation strategies fail to address patient-specific nuances, particularly when baseline morphological norms such as optic nerve cup-to-disc ratios naturally vary across age, race, and gender. To address this, we introduce Equi-Agent, a multi-agent framework that shifts the paradigm from static model ensembling to dynamic trust calibration. Operating as an autonomous clinical board, our architecture synthesizes outputs from diverse foundation models by critically cross-examining their localized visual findings against demographic error priors and patient metadata. Extensive evaluations on the Harvard FairVision and Harvard-GDP datasets demonstrate that our framework exceeds the diagnostic F1-score of standalone foundation models and significantly narrows the disparity in false positive and false negative rates across marginalized subgroups. Furthermore, the system achieves state-of-the-art accuracy in longitudinal glaucoma progression prediction, demonstrating its robust adaptability to complex temporal clinical reasoning. Our findings establish LLM-driven contextual arbitration as a robust, interpretable pathway for integrating biased foundation models into equitable clinical workflows.}

\keywords{ophthalmic foundation models, fairness, calibration, multi-agent systems, retina imaging}

\section{Experimental Design and Evaluation}\label{sec:experiments}

We designed the experimental evaluation to test whether dynamic trust calibration can improve the reliability of ophthalmic vision foundation models without sacrificing aggregate diagnostic performance. The central hypothesis is that the reliability of a foundation model is not fixed globally, but varies as a function of disease task, imaging modality, patient metadata, subgroup-specific error patterns, model disagreement, and longitudinal context. Accordingly, Equi-Agent is evaluated not only against standalone foundation models, but also against static ensembles, post-hoc calibration methods, and fairness-aware training baselines.

All experiments are structured around four evaluation questions. First, we test whether standalone ophthalmic foundation models exhibit subgroup-specific reliability gaps despite strong aggregate performance. Second, we test whether Equi-Agent improves diagnostic performance and worst-group reliability compared with standalone models and conventional ensembles. Third, we test whether the proposed agentic arbitration mechanism is necessary by comparing Equi-Agent against non-agentic fusion, single-agent arbitration, and component ablations. Fourth, we test whether the same arbitration framework extends to longitudinal glaucoma progression prediction.

Demographic metadata is used only as contextual information for estimating model reliability under subgroup-specific failure modes. It is not used as a direct proxy for disease status. To evaluate this design choice, we include metadata-only, metadata-shuffled, and no-demographic ablations.

\subsection{Equi-Agent Arbitration Framework}\label{subsec:equi_agent_framework}

Equi-Agent receives as input patient imaging data, patient metadata, predictions from multiple base models, model confidence scores, localized visual evidence, and validation-derived subgroup error priors. The system is structured as a multi-agent arbitration framework. A disease evidence agent evaluates whether the visual findings support the candidate diagnosis. A demographic context agent assesses whether patient metadata changes the reliability of specific visual cues or model outputs. A bias auditor agent compares the current prediction context against subgroup-specific false-positive and false-negative patterns estimated from the validation set. A model disagreement agent identifies conflicts among base models and determines whether disagreement reflects uncertainty, modality-specific weakness, or subgroup-specific unreliability. For longitudinal tasks, a temporal reasoning agent evaluates whether changes across visits are clinically plausible. A final arbitration agent integrates these outputs into a calibrated prediction, confidence score, and explanation.

To reduce stochasticity and improve reproducibility, agent outputs are constrained to a structured schema. Each agent must return a fixed set of fields, including the evidence considered, the reliability concern identified, the recommended trust adjustment, and a confidence estimate. The arbitration agent is instructed to use only the provided model outputs, metadata, subgroup priors, and image-derived findings.

\subsection{Evaluation Metrics}\label{subsec:metrics}

We report both aggregate diagnostic performance and subgroup-specific reliability. Aggregate performance is measured using area under the receiver operating characteristic curve (AUROC), F1-score, balanced accuracy, sensitivity, specificity, and area under the precision-recall curve where appropriate. Fairness and subgroup reliability are measured using worst-group F1, worst-group AUROC, false-positive rate gap, false-negative rate gap, equal opportunity difference, equalized odds difference, subgroup expected calibration error, and subgroup Brier score.

For a demographic attribute $A$ with subgroups $g \in \mathcal{G}$, the false-positive rate gap is defined as
\begin{equation}
\Delta \mathrm{FPR} = \max_{g \in \mathcal{G}} \mathrm{FPR}_{g} - \min_{g \in \mathcal{G}} \mathrm{FPR}_{g}.
\label{eq:fpr_gap}
\end{equation}

The false-negative rate gap is defined as
\begin{equation}
\Delta \mathrm{FNR} = \max_{g \in \mathcal{G}} \mathrm{FNR}_{g} - \min_{g \in \mathcal{G}} \mathrm{FNR}_{g}.
\label{eq:fnr_gap}
\end{equation}

Worst-group F1 is defined as
\begin{equation}
\mathrm{F1}_{\mathrm{worst}} = \min_{g \in \mathcal{G}} \mathrm{F1}_{g}.
\label{eq:worst_group_f1}
\end{equation}

For calibration, we report expected calibration error both globally and within each subgroup. Subgroup ECE is computed by binning predictions within each subgroup and measuring the weighted difference between confidence and empirical accuracy. We also report reliability diagrams, Brier score, and risk-coverage curves when abstention or uncertainty-aware decision-making is evaluated.

\subsection{Experiment 6: Metadata Safety and Counterfactual Sensitivity}\label{subsec:exp6}

The goal is to verify that metadata does not function as a direct disease shortcut. We compare image-only models, metadata-only models, naive image-plus-metadata models, Equi-Agent without metadata, Equi-Agent with shuffled metadata, and full Equi-Agent.

We also perform counterfactual metadata sensitivity tests. In these tests, image data and base-model outputs are held fixed while demographic metadata is changed. A robust arbitration system should not casually flip disease predictions solely because metadata changes. Instead, demographic context should primarily affect model trust weights, uncertainty estimates, or escalation decisions when subgroup-specific validation evidence supports such adjustment.

\subsection{Statistical Analysis}\label{subsec:statistical_analysis}

All metrics are reported with 95\% confidence intervals estimated by nonparametric bootstrapping over patients. When both eyes from the same patient are present, bootstrapping is performed at the patient level to avoid treating correlated eyes as independent samples. Comparisons between models are performed using paired bootstrap tests on the same test cases. For AUROC comparisons, DeLong's test may be used as a supplementary analysis. For subgroup metrics, we report subgroup sample sizes and confidence intervals to avoid overinterpreting unstable estimates from small groups.

Primary statistical comparisons are made between Equi-Agent and the strongest validation-selected non-agentic baseline. Secondary comparisons are made against the best standalone foundation model, the best static ensemble, and the best fairness-aware baseline. Multiple comparisons are controlled using the Benjamini--Hochberg procedure when reporting large numbers of subgroup-level tests.

\subsection{Planned Acceptance Criteria}\label{subsec:acceptance_criteria}

We define success as an improvement in worst-group reliability without unacceptable loss of aggregate diagnostic performance. Specifically, Equi-Agent is considered successful if it reduces false-positive or false-negative disparities relative to the strongest non-agentic baseline while maintaining comparable or improved AUROC and F1-score. For longitudinal glaucoma progression, Equi-Agent is considered successful if it improves subgroup reliability and achieves competitive temporal prediction performance. Claims of state-of-the-art performance are made only if Equi-Agent outperforms the strongest reproduced longitudinal baselines under the same test split and evaluation protocol.
